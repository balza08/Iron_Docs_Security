# Iron_Docs_Security
Welcome to **Iron Docs Security**, a platform designed for the secure management, storage, and consultation of digital documentation. This project was created with the goal of ensuring the integrity, confidentiality, and availability of sensitive data.

## Key Features
* **Secure Document Management:** Encryption of files both at rest and in transit.
* **Role-Based Access Control (RBAC):** Permission management based on user roles (Admin, Editor, Viewer).
* **Audit Log:** Complete tracking of all operations performed on documents.
* **Intuitive Interface:** A clean, fast, and responsive user experience.
